# TBH Inventory Editor 🛠️

Uma ferramenta utilitária para visualização e edição local de inventários. Este projeto foi desenvolvido exclusivamente para fins educacionais, análise de dados de jogo, depuração de desenvolvimento de modificações (modding) e backup de arquivos salvos (*savegames*) locais.

---

## ⚠️ Isenção de Responsabilidade (Disclaimer) & Política de Uso

Este projeto **não é um cheat (trapaça), hack, bot ou ferramenta de automação**.
- **Sem Vantagem Competitiva:** Não concede vantagens injustas em ambientes multijogador nem interfere em servidores online.
- **Respeito aos Direitos Autorais:** O repositório **não** distribui nenhum arquivo proprietário, binários do jogo ou dados protegidos por direitos autorais (conforme definido no arquivo [.gitignore](file:///C:/Dev/test-inv-view/.gitignore)).
- **Segurança de Conta:** O uso desta ferramenta é de inteira responsabilidade do usuário final e destina-se apenas a modificações em saves locais/offline.
- **Conformidade:** Este projeto está em total conformidade com as diretrizes da comunidade do GitHub para ferramentas de desenvolvimento de código aberto e modding de jogos.

---

## 🚀 Funcionalidades

- **Visualização de Inventário:** Interface limpa para analisar itens estruturados em arquivos locais.
- **Edição de Estruturas:** Edição direta de valores locais para testes de compatibilidade de itens.
- **Backup Seguro:** Permite gerenciar seus arquivos salvos sem risco de corrupção acidental.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5 & CSS3** para a interface do usuário de visualização e edição.
- **JavaScript (Vanilla)** para a lógica local de parse e manipulação dos dados.

---

## 📦 Como Usar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/matheus-fsc/TBH---Inventory-editor-.git
   ```
2. **Abra a ferramenta:**
   Execute o arquivo `run_editor.bat` ou abra `editor.html` diretamente no seu navegador.
3. **Carregue seus dados:**
   Importe o arquivo de salvamento do jogo (localizado localmente no seu computador) para começar a visualizar/editar.

---

## 🔒 Privacidade e Segurança

Todos os dados são processados **localmente no seu navegador**. Nenhum dado de jogo ou arquivo pessoal é enviado a servidores externos, garantindo total privacidade e segurança.
